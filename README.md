##chatGPT Clone, With the Name Of LiquidGPT.


LiquidGPT is a ChatGPT clone, for SMIT Hackathon built with React, Vite, and TailwindCSS v4. The application integrates with OpenRouter's API to provide multi-model AI chat capabilities with conversation management, dark mode, and a modern UI. The API key is also being secured with enviormental Variable.
